
# Project 1: Interactive Bill Manager

## Summary

Create a command line bills/expenses manager that runs interactively. This mini project brings together many of the concepts learned thus far into a single application.

## User Stories

### Stage 1
- I want to add bills, including the name and amount owed.
- I want to view existing bills.

### Stage 2
- I want to remove bills.

### Stage 3
- I want to edit existing bills.
- I want to go back if I change my mind.
